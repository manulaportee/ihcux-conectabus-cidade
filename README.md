# Contexto de Uso

O aplicativo foi projetado para situações do cotidiano urbano em que o usuário precisa se locomover rapidamente utilizando transporte público.  
Um dos principais cenários considerados foi:

> “Correndo para não perder o ônibus/metrô enquanto tenta acompanhar horários, lotação e localização em tempo real.”

O app também atende usuários que:
- Precisam verificar rapidamente a melhor rota;
- Desejam acompanhar a chegada do ônibus em tempo real;
- Utilizam carteira digital para pagamento;
- Precisam de informações acessíveis durante o trajeto;
- Estão em ambientes movimentados e com pouco tempo para navegar no sistema.

# Decisões de UX

Durante o desenvolvimento do protótipo, priorizamos informações essenciais para tomada rápida de decisão do usuário.

## Informações destacadas
- **Tempo de chegada do ônibus**
- **Lotação do veículo**
- **Rotas mais rápidas**
- **Mapa em tempo real**
- **Recursos do veículo (Wi-Fi, acessibilidade, ar-condicionado)**

Essas informações receberam maior destaque visual porque impactam diretamente na experiência e urgência do usuário durante o deslocamento.

## Informações menos prioritárias
Elementos secundários, como histórico de transações ou detalhes adicionais da rota, foram organizados em áreas separadas da interface para evitar excesso de informação e reduzir a carga cognitiva.

## Organização visual
As telas foram estruturadas com:
- Navegação simples;
- Ícones intuitivos;
- Poucos elementos por tela;
- Hierarquia visual clara;
- Destaque para ações rápidas.

O objetivo foi tornar a interação mais prática, rápida e intuitiva.

# Acessibilidade

O protótipo inclui recursos voltados para usuários com mobilidade reduzida e pessoas em situação de pressa.

## Para pessoas com mobilidade reduzida
- Indicação de veículos acessíveis com rampa;
- Informações rápidas sobre infraestrutura do transporte;
- Interface simples com botões grandes e organizados;
- Menor necessidade de múltiplos cliques.

## Para usuários com pressa
- Tempo de chegada destacado em tamanho maior;
- Lotação exibida com cores e etiquetas rápidas;
- Busca rápida de destino;
- Rotas recomendadas automaticamente;
- Acompanhamento em tempo real pelo mapa.

Essas decisões ajudam o usuário a encontrar informações importantes rapidamente, reduzindo esforço e aumentando a eficiência da navegação.

# Objetivo do Projeto

O objetivo do aplicativo é melhorar a experiência de mobilidade urbana, oferecendo informações rápidas, acessíveis e organizadas para auxiliar usuários durante seus deslocamentos diários.
